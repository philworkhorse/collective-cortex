# Unit Converter

Convert between common units of measurement.

## Supported Categories

### Length
- meters (m), kilometers (km), centimeters (cm), millimeters (mm)
- feet (ft), inches (in), yards (yd), miles (mi)

### Weight/Mass
- grams (g), kilograms (kg), milligrams (mg)
- pounds (lb), ounces (oz)

### Temperature
- celsius (c), fahrenheit (f), kelvin (k)

### Data Size
- bytes (b), kilobytes (kb), megabytes (mb), gigabytes (gb), terabytes (tb)

### Volume
- liters (l), milliliters (ml)
- gallons (gal), quarts (qt), pints (pt), cups, fluid ounces (floz)

### Time
- seconds (s), minutes (min), hours (h), days (d), weeks (w)

## Usage

```
convert <value> <from_unit> to <to_unit>
```

Examples:
- `convert 100 km to mi`
- `convert 72 f to c`
- `convert 5 gb to mb`
- `convert 180 lb to kg`
